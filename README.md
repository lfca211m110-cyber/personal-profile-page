# Personal Profile Page

This is my first personal profile website built as part of my journey to become a full-time web developer.

## Live Website

View the live website here:

[My Personal Profile Page](https://lfca211m110-cyber.github.io/personal-profile-page/)

## About the Project

This project is a beginner-friendly portfolio page created with HTML, CSS, and JavaScript. It includes basic webpage structure, styling, responsive design, and simple JavaScript interactions.

## Features

- Navigation menu
- Hero section with profile image
- Project cards
- Contact form
- Form validation
- Motivation button
- Dark mode toggle
- Responsive mobile design

## Technologies Used

- HTML
- CSS
- JavaScript
- Git
- GitHub
- GitHub Pages
